# 総合試験仕様書（手動）

紅白予想アプリの機能を、実際に操作して期待どおり動くか・不具合がないかを確認するための手順書。
各ケースの「合否」欄に ✅/❌ を記入し、❌ は「メモ」に状況を残す。

- 凡例: 〔要OAuth〕= Googleログイン設定が必要 / 〔要Solana〕= devnet mint・RPC・鍵が必要
- API のエラー形式は常に `{ "error": { "code": ..., "message": ... } }`

---

## 0. 事前準備

### 0.1 ローカル起動

```bash
# API（server）
cd server
pnpm db:migrate:local      # スキーマ適用
pnpm db:seed:local         # サンプルのシーズン・アーティスト・曲
pnpm dev                   # http://localhost:8787

# フロント（web）。別ターミナル
cd web
pnpm dev                   # http://localhost:5173
```

- 〔要OAuth〕`server/.dev.vars` を `.dev.vars.example` を元に作成し、Google OAuth クライアントの
  値と `SESSION_SECRET` を設定。Google Console のリダイレクトURIに
  `http://localhost:5173/api/auth/google/callback` を登録。
- ブラウザで http://localhost:5173 を開く。

### 0.2 自動テスト（参考・前提確認）

```bash
cd server && pnpm test      # 全ロジック/APIの自動テストが緑であること
```

| No  | 確認内容       | 期待結果                                                     | 合否 | メモ |

| 0-2 | アプリ初期表示 | ヘッダ「紅白予想」とタブ（予想/ランキング/ウォレット）が出る |      |      |

## 4. 管理機能 〔要OAuth〕

前提: 自分を管理者にする。

```bash
cd server
wrangler d1 execute kohaku-db --local \
  --command "UPDATE users SET is_admin=1 WHERE email='あなたのGmail';"
```

API は curl 等で叩く（管理画面は未実装）。Cookie が必要なので、ブラウザのセッションを使うか
`--cookie` を付与する。

| No   | 確認内容                   | 操作                                               | 期待結果                         | 合否 | メモ |
| ---- | -------------------------- | -------------------------------------------------- | -------------------------------- | ---- | ---- |
| 4-1  | 非管理者ガード             | is_admin=0 のユーザーで `POST /api/admin/seasons`  | 403 `FORBIDDEN`                  |      |      |
| 4-2  | シーズン作成               | `POST /api/admin/seasons` `{"year":2027}`          | 201・作成される                  |      |      |
| 4-3  | アーティスト作成（別名）   | `POST /api/admin/artists`（name + aliases）        | 201、別名検索でヒット            |      |      |

## 7. 記念トークン配布・表示 〔要OAuth〕〔要Solana〕

を設定（devnet）。treasury に SOL。当選ユーザーはウォレット連携済み。

| No  | 確認内容         | 操作                                                | 期待結果                                     | 合否 | メモ |
| --- | ---------------- | --------------------------------------------------- | -------------------------------------------- | ---- | ---- |

| 7-2 | 締切前           | 受付中シーズンで distribute                         | 409 `CONFLICT`                               |      |      |
